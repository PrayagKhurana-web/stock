# Stock Analysis Bot (Streamlit Version)

A machine-learning powered stock analysis tool with:

- Data fetching using Yahoo Finance
- Technical indicators (RSI, MACD, Bollinger Bands, etc.)
- ML Models (RandomForest + XGBoost)
- Predictions for next N days
- Backtesting (ML Strategy vs Buy & Hold)
- Candlestick charts with Plotly
- Telegram Alerts
- Portfolio / Paper Trading Simulation
- Streamlit-based Web UI
